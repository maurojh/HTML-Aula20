Entrada Acesse o site: https://dartpad.dartlang.org/. Digite o código abaixo em DART: void main() { print("Hello Barretos!"); } Para executar o código aperte o botão RUN. O programa irá exibir em CONSOLE: Hello Barretos! Usando variáveis: Digite o código abaixo em DART: void main() { var nome = "Fulano"; print("Hello $nome!"); } A variável nome receberá o valor "Fulano" e será usada em print.
